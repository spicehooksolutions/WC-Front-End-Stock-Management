Quickly manage WooCommerce Product Stoks, Stock Status , featured image quickly from Front End. Accessible for Administrator and Shop Manager only.

== Description ==
WC Front End Stock Management - gives you ( Administrator / Shop Manager) to quickly manage the WooCommerce Product Stock Status, Product Stock Quantity, Product Featured Image, Product Categories assignment without entered in WordPress Administration area. 

== Installation ==
1. Unzip the downloaded .zip file.
2. Upload the plugin folder to the wp-content/plugins/ directory of your WordPress site.
3. Go to the 'Plugins' menu in WordPress and activate the Smart WooCommerce Search plugin.
4. Configure the plugin settings (/wp-admin/admin.php?page=woocommerce-front-end-stock-management)
5. Place the shortcode [wfesm_stock_management] on the desired page.
